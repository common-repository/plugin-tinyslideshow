Simple Tiny SlideShow

== Description ==

This plugin is a JavaScript slide show is full of features. 
Features include smooth animation, title / description / link support, 
no naming restrictions, graceful degradation, on demand loading of image and status of assets.

This plugin implements the JavaScript slideshow is from Mechael Leibeger.


== Installation ==
