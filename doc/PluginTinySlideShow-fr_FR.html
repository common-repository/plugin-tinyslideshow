Aide Plugin TinySlideShow
  • Généralité

    Ce Plugin est basé sur le script TinySlideShow de Michael Leigeber (www.leigeber.com).
    Créez une nouvelle galerie dans le répertoire ' galleries ' du plugin. Créez en suite le répertoire thumbnails,
    il doit impérativement est nommé comme indiqué.
    Téléchargez vos images ou photos, le nom du répertoire est utilisé pour identifié la galerie.
    Créez toutes les galeries que vous souhaitez, supprimez et modifiez.

    Utilisez le shortcode pour chaque galerie pour l'insérer dans vos pages ou posts.

  • Paramétrage

    • Titre: donne un titre au TinySlider, sinon laisser vide pour aucun titre.
    • Vitesse: interval du diaporama automatique.
      • 0 (zéro) : désactivé
      • 1..10: 1 étant le plus rapide
    • Galerie: le répertoire contenant les images ou photos à afficher.

      Pour chaque image ou photo de la galerie définissez le titre et la description.

    • Vitesse défilement: vitesse de défillement du Thumbnails (cf. Vitesse).
    • Automatique: Oui ou Non, slide automatique.
    • Espacement: espace entre les éléments
    • Active: couleur de l'élément actif du Thumbnails

  • Ajouter, Supprimer, Modifier

    • Cette page vous permet de gérer l'ensemble des TinySlideShow enregistrés. vous pouvez supprimer plusieurs TinySlideShows en les cochants dans la liste et click sur ' Supprimer '.

  • Utilisation

    • Copier le ' Shortcode ' dans la liste du TinySlideShow souhaité et copier dans la page ou l'article cible.

  • Astuce

    • Pour personnaliser l'affichage du TinySlideShow, modifier le fichier tinyslideshow.css contenu dans le répertoire css du plugin.
    Pour optimiser le diaporama n'utilisez pas d'image ou de photo trop grande, et modifier les dimensions du TinySlider dans le css afin qu'il soit le plus adapté aux dimensions des images.

  • Support

    • Pour suggérer une évolution, amélioration ou signaler une anomalie utilisez le formulaire contact en précisant 'WP TinySlideShow' dans l'objet.
